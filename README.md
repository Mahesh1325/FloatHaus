# Sensory Deprivation Float Center Template

A premium, modern, and highly responsive HTML template designed specifically for Sensory Deprivation Float Centers and luxury wellness spas. Features a calming aesthetic, comprehensive multi-step booking logic, responsive layout, dark/light themes, and a robust set of static pages.

## Features

- **Dark Mode by Default**: Tailored to impart a feeling of calm and rest.
- **RTL Support**: Built-in `dir="rtl"` functionality.
- **Fully Responsive**: Mobile-first design that scales perfectly to tablets and desktop displays.
- **Interactive Booking Flow**: A multi-step form built with vanilla JavaScript simulating a comprehensive scheduling system.
- **Dashboard Interface**: Includes toggles between User and Admin views to demonstrate application logic.
- **Accessibility Ready**: Built to WCAG 2.1 AA standards.

## Included Pages

- `index.html` (Home)
- `about.html` (About the center)
- `float-therapy.html` (Therapy Details and Services)
- `memberships.html` (Pricing tables)
- `booking.html` (Multi-step interactive form)
- `confirmation.html` (Success screen)
- `dashboard.html` (User and Admin views)
- `contact.html` (Contact and Map)
- `coming-soon.html` (Lead Generation)
- `404.html` (Error Page)

## Quick Start

Simply open the `index.html` file in your browser. Since this is an HTML/CSS/JS template built without a bundler, no installation steps (`npm install`) are strictly required. 

For full customization instructions, please refer to the comprehensive guide in the `documentation/` folder:

1. Open `documentation/index.html` in your browser.

## Customization Highlights

- **CSS Variables**: Easily change the primary brand colors by modifying the variables inside `:root` within `assets/css/style.css`.
- **Forms**: The contact form requires a backend like Formspree or similar to process submissions.
- **Payment Processing**: The booking form payment is illustrative. Integrations with Stripe Elements should replace the placeholder.
