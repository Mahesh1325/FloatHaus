Sensory Deprivation Float Center Template

Thank you for purchasing the Float Center template! This guide covers everything needed to customize and deploy this project.

Installation Guide

This is a static HTML template requiring no build steps, Node.js, or NPM. To view the template:

  1. Extract the provided ZIP file into your project folder.
  2. Open any .html file inside the pages/ directory directly in your web browser.
  3. If you are uploading to a server, simply place all folders (assets, pages) in the root directory or configure your routing appropriately.

If you prefer using a local server for testing, you can use VSCode's "Live Server" extension.

Customization Guide

All core customizations heavily rely on modifying HTML and Vanilla CSS. To edit content, open the respective HTML file (e.g., index.html) and edit the text directly between the tags.

How to Change Colors

Colors are controlled by CSS Variables. Open assets/css/style.css and look at the :root selector at the very top. 

:root {
  --primary: #4f46e5; /* Main Brand Color */
  --secondary: #0f172a; /* Navy Background */
  --accent: #818cf8; /* Soft Accents */
}

Simply replace these hex codes with your brand colors. The template will automatically update across all pages.

How to Edit Booking Slots

The time slots are dynamically rendered via vanilla JavaScript in the booking flow.

  1. Open assets/js/booking.js
  2. Locate the populateTimeSlots() function.
  3. Edit the times array to reflect your actual availability: 
    const times = ['8:00 AM', '10:00 AM', '12:00 PM', '2:00 PM', '4:00 PM'];

How to Enable RTL

This template supports Right-to-Left (RTL) languages inherently.

  1. In any HTML file, change the <html> tag's dir attribute to RTL:
    <html lang="ar" dir="rtl" data-theme="dark">
  2. Ensure assets/css/rtl.css is loaded on the page.

How to Integrate Stripe

The payment screen in booking.html is a visual placeholder. To make it functional:

  1. Create an account at Stripe.com.
  2. Include Stripe.js in your HTML: <script src="https://js.stripe.com/v3/"></script>
  3. Replace the .stripe-placeholder div inside booking.html with a container for Stripe Elements: 
    <div id="card-element"></div>
  4. Mount the Stripe element using custom JavaScript inside your checkout flow logic.
